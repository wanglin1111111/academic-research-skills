## 模板C：快速筛查清单（5分钟版）

### 致命缺陷筛查（任一通过即排除）
- [ ] 单一业务占比>70%？
- [ ] 投资收益/净利润>40%？
- [ ] 经营现金流/净利润<0.6？
- [ ] PE>40倍且历史分位>70%？
- [ ] 股东人数季度增长>30%？
- [ ] 季度盈利波动>50%？

### 核心指标检查
- [ ] 产业链自给率>30%？
- [ ] 供应商CR3<60%？
- [ ] 处于盈利爬坡期或收获期？
- [ ] 估值处于历史50%分位以下？
- [ ] 机构持股稳定或增加？

### 快速决策
- [ ] **绿灯** - 通过所有筛查，可深入研究
- [ ] **黄灯** - 部分指标存疑，谨慎对待
- [ ] **红灯** - 存在致命缺陷，建议回避
