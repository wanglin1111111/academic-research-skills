```
## 交互设计
### 唤醒机制
- 唤醒词："小虾" / "Shadow"
- 本地唤醒（端侧）
- 误唤醒率 < 1次/24小时

### 对话流程
```
用户说话
    ↓
[唤醒检测] → 未唤醒 → 继续监听
    ↓ 唤醒
[语音识别] → 文本
    ↓
[语义理解] → 意图 + 实体
    ↓
[对话管理] → 回复策略
    ↓
[语音合成] → 音频
    ↓
[音频播放] → 用户听到
```

### 多轮对话
- 上下文保持（最近 5 轮）
- 打断处理（用户可随时打断）
- 纠错机制（"不是，我是说..."）
- 沉默检测（长时间无回应自动结束）
```

## ⚙️ 技术实现要点

### 语音识别 Pipeline

```python
# 语音识别 Pipeline
class VoiceRecognitionPipeline:
    def __init__(self):
        self.vad = VoiceActivityDetector()
        self.noise_reducer = NoiseReducer()
        self.asr_engine = ASREngine()
        self.post_processor = PostProcessor()
    
    def process(self, audio_stream):
        """处理语音流"""
        # 1. 语音活动检测
        has_speech = self.vad.detect(audio_stream)
        if not has_speech:
            return None
        
        # 2. 降噪处理
        clean_audio = self.noise_reducer.reduce(audio_stream)
        
        # 3. 语音识别
        text = self.asr_engine.recognize(clean_audio)
        
        # 4. 后处理
        processed_text = self.post_processor.process(text)
        
        return processed_text

class VoiceActivityDetector:
    def detect(self, audio_chunk):
        """检测语音活动"""
        # 使用能量阈值 + 过零率
        energy = self.calculate_energy(audio_chunk)
        zcr = self.calculate_zcr(audio_chunk)
        
        return energy > self.threshold and zcr > self.zcr_threshold

class NoiseReducer:
    def reduce(self, audio):
        """降噪处理"""
        # 使用 spectral subtraction 或 deep learning 降噪
        return self.spectral_subtraction(audio)
```

### 情感化语音合成

```python
# 情感化语音合成
class EmotionalTTS:
    def __init__(self):
        self.emotion_models = {
            'neutral': NeutralEmotionModel(),
            'happy': HappyEmotionModel(),
            'sad': SadEmotionModel(),
            'excited': ExcitedEmotionModel(),
            'calm': CalmEmotionModel()
        }
    
    def synthesize(self, text, emotion='neutral', intensity=0.5):
        """合成带情感的语音"""
        # 1. 文本分析
        text_analysis = self.analyze_text(text)
        
        # 2. 情感参数提取
        emotion_params = self.get_emotion_params(emotion, intensity)
        
        # 3. 韵律生成
        prosody = self.generate_prosody(text_analysis, emotion_params)
        
        # 4. 语音合成
        audio = self.tts_engine.synthesize(text, prosody)
        
        return audio
    
    def get_emotion_params(self, emotion, intensity):
        """获取情感参数"""
        params = {
            'happy': {
                'pitch_shift': 1.1,      # 音调升高
                'speed': 1.05,            # 语速稍快
                'energy': 1.2,            # 音量增大
                'variability': 0.3        # 韵律变化增加
            },
            'sad': {
                'pitch_shift': 0.9,      # 音调降低
                'speed': 0.9,             # 语速减慢
                'energy': 0.8,            # 音量减小
                'variability': 0.2        # 韵律变化减少
            },
            'calm': {
                'pitch_shift': 1.0,      # 音调平稳
                'speed': 0.95,            # 语速适中
                'energy': 0.9,            # 音量适中
                'variability': 0.15       # 韵律平稳
            }
        }
        
        base_params = params.get(emotion, params['neutral'])
        # 根据强度调整参数
        return {k: 1 + (v - 1) * intensity for k, v in base_params.items()}
```

## ⚠️ 注意事项

1. **隐私保护**：语音数据高度敏感，需严格保护
2. **场景适配**：不同场景对延迟、准确率要求不同
3. **成本控制**：语音 API 成本可能很高
4. **用户体验**：语音交互的自然度是核心竞争力
5. **离线能力**：碎片场景需要考虑离线可用性
