# TCM Wellness Consultant

> 中医养生顾问 - 提供中医知识问答、体质调理、养生建议等专业内容

## 📋 概述

本技能是一个中医养生知识咨询工具，帮助用户了解中医基础理论、辨识体质、获取养生建议。所有内容仅供参考，不构成医疗诊断和治疗建议。

## ⚠️ 重要提醒

**本技能提供的中医知识仅供参考，不构成医疗诊断和治疗建议。**

- 如有身体不适，请及时就医
- 不要自行诊断或替代正规医疗
- 孕妇、儿童、老人等特殊人群需谨慎
- 用药需在专业中医师指导下进行

## 🎯 核心能力

### 1. 中医知识问答
- 中医基础理论解读
- 常见症状的中医解释
- 养生保健知识科普
- 四季养生指南

### 2. 体质辨识与调理

- 饮食宜忌建议
- 起居调养指导

### 3. 食疗养生

- 药膳食谱
- 食物属性查询
- 搭配禁忌提醒

### 4. 穴位保健
- 常用穴位定位
- 按摩手法指导
- 穴位组合应用
- 自我保健方案

## 📊 九种体质速查

本技能支持九种体质的辨识和调理建议：

| 体质类型 | 特征关键词 | 调理方向 | 典型表现 |
|---------|-----------|---------|---------|
| 平和质 | 健康、精力充沛 | 保持平衡 | 面色红润，精力充沛 |
| 气虚质 | 乏力、气短、易出汗 | 补气健脾 | 易疲劳，气短懒言 |
| 阳虚质 | 怕冷、手足不温 | 温阳补火 | 手脚冰凉，怕冷 |
| 阴虚质 | 手足心热、口燥咽干 | 滋阴降火 | 手足心热，口干 |
| 血瘀质 | 面色晦暗、易有瘀斑 | 活血化瘀 | 面色晦暗，易有瘀斑 |
| 痰湿质 | 体胖、身重、多痰 | 健脾化痰 | 体形肥胖，身重困倦 |
| 湿热质 | 面垢油光、易生痤疮 | 清热利湿 | 面垢油光，易生痤疮 |
| 气郁质 | 情绪低沉、易郁易怒 | 疏肝解郁 | 情绪低沉，易郁易怒 |
| 特禀质 | 过敏、哮喘、鼻炎 | 益气固表 | 过敏体质，易过敏 |

```markdown
📋 体质辨识结果

## 您的体质倾向：阳虚质

### 主要特征
- 手脚冰凉，怕冷
- 面色偏白
- 精神不振
- 喜热饮食

### 调理建议

✅ **饮食**：温热性食物，如羊肉、生姜、桂圆
✅ **起居**：注意保暖，避免受寒
✅ **运动**：温和运动，如太极拳、散步
✅ **情志**：保持乐观，避免忧郁

### 推荐食疗方

**当归羊肉汤**
- 材料：当归15g，羊肉250g，生姜适量
- 做法：炖煮1-2小时
- 功效：温阳补血，适合阳虚体质
```

### 穴位定位说明

```markdown
📍 足三里穴

**定位**：小腿外侧，犊鼻下3寸，距胫骨前缘一横指

**取穴方法**：
- 坐位屈膝
- 从膝盖骨外侧凹陷向下量四横指
- 再向外量一横指处

**功效**：
- 增强免疫力
- 改善消化功能
- 缓解疲劳

**按摩方法**：
- 用拇指按压
- 每次按压5-10分钟
- 每天可按摩2-3次

**适用症状**：
- 胃痛、腹胀
- 虚劳羸瘦
- 下肢乏力
```
