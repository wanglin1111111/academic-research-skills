# 技能应用指南 (Skill Application Guide)

## 快速开始

### 步骤1：明确分析目标
在使用本技能前，先明确：
- 分析对象（股票代码、公司名称）
- 分析目的（投资决策、研究跟踪、风险评估）
- 时间维度（短期交易、中期持有、长期投资）

### 步骤2：数据收集
按照以下清单收集数据：

```
必备数据源：
□ 公司公告（定增、并购、业绩预告）
□ 财务报告（年报、季报）
□ 行业研报（券商研报、行业数据）
□ 市场数据（股价、成交量、技术指标）
□ 政策文件（产业政策、监管规定）
```

### 步骤3：系统化分析
按照技能框架的8个维度逐一分析：
1. 基本面分析 → 2. 技术面分析 → 3. 竞品对比 → 4. 政策分析 → 5. 资本运作 → 6. 辩证评估 → 7. 投资策略 → 8. 跟踪计划

### 步骤4：输出报告
按照标准模板输出分析报告，确保：
- 逻辑清晰
- 数据准确
- 观点明确
- 风险提示充分

## 场景化应用

### 场景1：新股申购/上市分析

**重点模块**：
- 基本面分析（盈利能力、成长性）
- 行业分析（赛道景气度）
- 竞品对比（估值对标）

**关键指标**：
- 发行PE vs 行业平均PE
- 募资用途合理性
- 网下申购热度

### 场景2：定增项目评估

**重点模块**：
- 资本运作分析（定增方案）
- 基本面分析（业绩承诺）
- 辩证评估（稀释风险 vs 成长机会）

**关键指标**：
- 发行价格折扣率
- 锁定期安排
- 募资用途与主业协同度

### 场景3：行业龙头比较

**重点模块**：
- 竞品对比（多维度对标）
- 竞争优势评估
- 投资定位

**关键指标**：
- 市场份额变化
- ROE对比
- 估值差异分析

### 场景4：事件驱动交易

**重点模块**：
- 技术面分析（趋势、成交量）
- 政策分析（事件影响）
- 短期策略

**关键指标**：
- 事件催化剂强度
- 技术面突破确认
- 资金流向

### 场景5：长期价值投资

**重点模块**：
- 基本面分析（护城河、成长性）
- 辩证评估（优势与风险）
- 长期策略

**关键指标**：
- 自由现金流
- 护城河深度
- 估值安全边际

## 常见问题

### Q1：如何快速判断一家公司是否值得深入研究？

**快速筛选 checklist**：
- [ ] 营收增速 > 15%
- [ ] ROE > 10%
- [ ] 毛利率稳定或提升
- [ ] 行业景气度向上
- [ ] 估值相对合理（PE < 行业平均或历史均值）

**3分钟快速评估法**：
1. 看赛道：是否处于高景气行业？
2. 看地位：是否为细分龙头？
3. 看业绩：近3年营收/利润是否持续增长？
4. 看估值：PE/PB是否处于合理区间？

### Q2：技术面和基本面如何结合？

**结合框架**：
```
基本面决定方向 → 技术面决定时机

强势基本面 + 强势技术面 = 重仓买入
强势基本面 + 弱势技术面 = 等待回调
弱势基本面 + 强势技术面 = 短线交易
弱势基本面 + 弱势技术面 = 回避
```

### Q3：如何处理信息过载？

**优先级排序**：
1. **必须关注**：业绩公告、重大合同、政策变化
2. **重要关注**：行业数据、竞品动态、机构评级
3. **一般关注**：市场传闻、短期波动、技术细节

**信息过滤原则**：
- 关注影响业绩的实质性信息
- 忽略噪音和短期波动
- 聚焦可验证的数据

### Q4：如何评估分析质量？

**自检清单**：
- [ ] 数据来源是否可靠？
- [ ] 逻辑链条是否完整？
- [ ] 是否考虑了反面证据？
- [ ] 结论是否有数据支撑？
- [ ] 风险提示是否充分？

### Q5：如何应对分析错误？

**纠错机制**：
1. **建立预警**：设置关键指标阈值
2. **定期复盘**：季度回顾分析准确性
3. **快速止损**：触发预警时果断行动
4. **总结教训**：记录错误原因，持续改进

## 进阶技巧

### 技巧1：多维度交叉验证

**验证方法**：
- 财务数据 vs 业务数据（收入 vs 订单）
- 公司公告 vs 行业数据（增速 vs 行业平均）
- 管理层表述 vs 实际业绩（承诺 vs 兑现）

### 技巧2：建立比较基准

**基准选择**：
- 行业平均：与同业公司对比
- 历史表现：与公司历史数据对比
- 市场预期：与券商预测对比
- 国际对标：与海外龙头对比

### 技巧3：情景分析应用

**三步法**：
1. **基准情景**：最可能发生的情况（概率50%）
2. **乐观情景**：超预期情况（概率30%）
3. **悲观情景**：风险情景（概率20%）

**投资决策**：
- 基准情景下：持有/买入
- 乐观情景下：加仓
- 悲观情景下：减仓/止损

### 技巧4：构建跟踪系统

**自动化工具**：
- 股价预警（同花顺、东方财富）
- 公告提醒（巨潮资讯网）
- 研报跟踪（Wind、慧博投研）
