# 投资分析框架详解

## 一、产业链分析框架

### 1.1 产业链自给率分析

**适用行业**：制造业、新能源、半导体、汽车等

**分析步骤**：
1. **识别产业链环节**
   - 上游：原材料、零部件
   - 中游：制造、组装
   - 下游：销售、服务

2. **评估自给率**
   ```
   自给率 = 自产数量 / 总需求数量 × 100%
   
   风险等级：
   - <30%：亮黄灯（高度依赖外部）
   - 30%-50%：关注（存在一定风险）
   - 50%-70%：良好（基本可控）
   - >70%：优秀（自主可控）
   ```

3. **供应商集中度分析**
   ```
   CR3 = 前三大供应商采购额 / 总采购额 × 100%
   
   风险等级：
   - >60%：高风险
   - 40%-60%：中等风险
   - <40%：低风险
   ```

### 1.2 盈利周期定位

**四个阶段特征**：

| 阶段 | 特征 | 投资策略 |
|------|------|----------|
| 概念期 | 技术验证，无收入 | 观望 |
| 投入期 | 资本开支大，亏损 | 谨慎 |
| 爬坡期 | 产能释放，盈利改善 | 布局 |
| 收获期 | 盈利稳定，现金流好 | 持有 |

## 二、业务结构分析框架

### 2.1 多元化程度评估

**单一业务占比**：
```
风险等级：
- >70%：红灯（过度集中）
- 50%-70%：黄灯（需关注）
- 30%-50%：良好（适度多元）
- <30%：绿灯（充分多元）
```

**业务协同性评估**：
- 强协同：共享客户、渠道、技术
- 弱协同：各自独立运营
- 负协同：资源分散，管理复杂

### 2.2 盈利质量分析

**毛利率对比**：
```
各业务毛利率差异分析：
- 差异>20个百分点：需深入分析原因
- 低毛利业务占比>60%：盈利质量堪忧
- 高毛利业务增长：积极信号
```

**盈利稳定性检验**：
```
季度盈利波动率 = 标准差(季度净利润) / 平均净利润 × 100%

风险等级：
- >50%：红灯（高度不稳定）
- 30%-50%：黄灯（波动较大）
- <30%：绿灯（相对稳定）
```

### 2.3 股东结构分析

**散户化指标**：
```
散户化程度 = 散户数量变化 / 总股东数量 × 100%

信号解读：
- >30%：机构撤退信号
- 15%-30%：需关注
- <15%：机构稳定
```

## 三、收益质量分析框架

### 3.1 利润结构拆分

**核心公式**：
```
净利润 = 经营利润 + 投资收益 + 营业外收支 - 所得税

收益质量指标：
- 经营利润占比 = 经营利润 / 净利润 × 100%
- 投资收益占比 = 投资收益 / 净利润 × 100%
- 营业外收支占比 = 营业外收支 / 净利润 × 100%
```

**风险等级**：
| 指标 | 红灯 | 黄灯 | 绿灯 |
|------|------|------|------|
| 投资收益占比 | >40% | 20%-40% | <20% |
| 经营利润占比 | <60% | 60%-80% | >80% |

### 3.2 现金流质量检验

**关键比率**：
```
经营现金流/净利润 = 经营现金流净额 / 净利润

解读：
- >1.0：盈利质量优秀
- 0.8-1.0：盈利质量良好
- 0.6-0.8：需关注
- <0.6：红灯（利润含金量低）
```

### 3.3 估值方法选择

**不同行业适用方法**：

| 行业类型 | 推荐方法 | 不适用方法 |
|----------|----------|------------|
| 基础设施 | DCF、股息率 | PE |
| 周期股 | PB、EV/EBITDA | PE |
| 成长股 | PEG、PS | 单纯PE |
| 金融股 | PB、PE | EV/EBITDA |
| 科技股 | PS、PEG | PB |

## 四、成本敏感度分析框架

### 4.1 周期定位

**四周期模型**：
```
复苏期：需求回升，产能利用率提升
繁荣期：供需紧张，价格上涨
衰退期：需求下滑，库存积压
萧条期：产能出清，价格触底

判断指标：
- 产能利用率
- 库存周转天数
- 价格走势
- 资本开支意愿
```

### 4.2 成本敏感度量化

**原材料敏感度**：
```
原材料成本敏感度 = (原材料成本 / 总成本) × (价格弹性)

示例：航空燃油
- 燃油成本占比：30%
- 油价弹性：1.0（完全传导）
- 敏感度：30%
- 油价每涨10% → 成本增加 = 营收 × 30% × 10%
```

**汇率敏感度**：
```
汇率敏感度 = 外币负债 / 总资产 × 汇率弹性

示例：航空公司
- 外币负债占比：40%
- 汇率弹性：1.0
- 敏感度：40%
- 人民币贬值1% → 汇兑损失 = 总资产 × 40% × 1%
```

### 4.3 经营杠杆测算

**经营杠杆系数**：
```
DOL = (营业收入 - 变动成本) / (营业收入 - 变动成本 - 固定成本)
    = 边际贡献 / 经营利润

解读：
- DOL > 2：高经营杠杆（收入±10% → 利润±20%以上）
- DOL 1.5-2：中等经营杠杆
- DOL < 1.5：低经营杠杆
```

## 五、主题投资风险评估框架

### 5.1 信息来源评估

**可信度排序**：
1. 独立研究报告（券商、咨询机构）
2. 公司公告、财报
3. 行业数据（统计局、行业协会）
4. 媒体报道（需交叉验证）
5. 网友/社交媒体（风险最高）

### 5.2 估值安全垫评估

**历史分位法**：
```
估值分位 = 当前估值在历史区间中的位置

风险等级：
- >70%分位：红灯（高估）
- 50%-70%分位：黄灯（偏高）
- 30%-50%分位：合理
- <30%分位：绿灯（低估）
```

**安全边际计算**：
```
安全边际 = (内在价值 - 市场价格) / 内在价值 × 100%

建议：
- 主题投资：安全边际 > 30%
- 价值投资：安全边际 > 50%
```

### 5.3 不确定性量化

**三重不确定性评估**：

| 类型 | 评估维度 | 量化方法 |
|------|----------|----------|
| 地缘风险 | 冲突概率、持续时间 | 情景分析 |
| 技术风险 | 突破时间、商业化前景 | 专家评估 |
| 政策风险 | 持续性、力度变化 | 政策跟踪 |

## 六、反向验证框架

### 6.1 反向思考清单

**利好确认时的反向验证**：
```
1. 这个利好是否已经被市场充分定价？
2. 是否存在被忽视的潜在利空？
3. 最坏情景下亏损多少？能否承受？
4. 是否有更好的替代标的？
5. 如果判断错误，退出策略是什么？
```

### 6.2 压力测试

**情景分析**：
```
基准情景：假设当前趋势延续
乐观情景：利好超预期兑现
悲观情景：出现重大不利变化
极端情景：黑天鹅事件

测试内容：
- 各情景下的盈利预测
- 估值变化
- 股价下跌空间
- 持仓承受能力
```

### 6.3 三重止损体系

| 止损类型 | 触发条件 | 执行方式 |
|----------|----------|----------|
| 技术止损 | 跌破关键支撑位 | 立即卖出 |
| 基本面止损 | 核心假设被证伪 | 评估后卖出 |
| 时间止损 | 持有期超预期未兑现 | 减仓或卖出 |
