# 多因子趋势分析框架

## 一句话总述

> 短期市场是投票机，长期是称重机——多因子框架帮你区分「情绪驱动的噪音」和「基本面驱动的信号」。

## 核心方法论：短长期分离

### 短期分析维度（噪音层）

| 因子 | 说明 | 典型信号 |
|------|------|---------|
| 交易拥挤度 | 仓位是否过度集中 | 某板块成交额占比超历史90%分位 |
| 获利盘压力 | 短期涨幅过大 | 某板块1个月涨超30% |
| 情绪指标 | 市场情绪是否过热 | 新增开户数暴增、融资余额骤升 |
| 事件扰动 | 短期催化/压制因素 | 单次数据超预期引发波动 |

**核心判断：** 短期调整如果只由以上因子驱动 → 是噪音，不改变长期趋势。

### 长期分析维度（信号层）

| 因子 | 说明 | 典型信号 |
|------|------|---------|
| 产业资本开支 | 行业龙头是否持续投入 | 头部企业Capex增速>20% |
| 盈利能力 | 产业链利润是否持续改善 | 毛利率/净利率连续3季上行 |
| 流动性环境 | 资金面是否支持 | 无风险利率下行或维持低位 |
| 结构性需求 | 需求是否趋势性扩张 | 某技术渗透率每年提升>5pct |

**核心判断：** 长期趋势如果由以上因子驱动 → 是信号，短期调整是布局窗口。

## 多因子评分表

对每个市场/板块，填写以下评分表（每因子1-5分，5分为最强）：

 
| 因子类型 | 因子名称 | 得分(1-5) | 判断依据 |
|---------|---------|-----------|---------|
| 短期-拥挤度 | 交易拥挤度 | ___ | _________ |
| 短期-获利盘 | 短期涨幅 | ___ | _________ |
| 短期-情绪 | 市场情绪 | ___ | _________ |
| 短期-事件 | 近期事件扰动 | ___ | _________ |
| 长期-资本开支 | 产业Capex趋势 | ___ | _________ |
| 长期-盈利能力 | 产业链利润 | ___ | _________ |
| 长期-流动性 | 资金环境 | ___ | _________ |
| 长期-结构性需求 | 需求扩张趋势 | ___ | _________ |
| **短期总分** | (满分20) | ___/20 | |
| **长期总分** | (满分20) | ___/20 | |
 `

**解读规则：**
- 短期高分(15-20) + 长期低分(<10) = 高风险，趋势可能反转
- 短期低分(<10) + 长期高分(15-20) = 好时机，调整是布局窗口
- 短期高分 + 长期高分 = 强势延续，但需注意短期回调风险
- 短期低分 + 长期低分 = 双弱，回避

## 噪音过滤四问

面对市场大幅波动时，依次问自己：

1. **「这次不一样吗？」** — 类似调整历史上有无先例？如果有，后续怎么走的？
2. **「基本面变了吗？」** — 产业资本开支、盈利能力、流动性有没有实质恶化？
3. **「谁在卖？为什么卖？」** — 是获利了结（健康）还是恐慌出逃（机会）？
4. **「3年后这件事还重要吗？」** — 如果答案是「不重要」，那就是噪音。

## 关键事件节点分析法

部分短期调整由「关键事件节点」触发（如央行行长讲话、重要经济数据发布）。

**处理框架：**
1. 列出未来30天内的关键事件节点（央行会议、数据发布、财报季等）
2. 对每个节点，预判三种情景：鹰派/中性/鸽派
3. 计算每种情景下的市场反应概率
4. 事件落地后，重新评估：是「预期差的修正」（短期）还是「趋势的改变」（长期）？

**重要原则：** 关键事件的影响通常是短期的，除非它改变了长期因子的评分。
