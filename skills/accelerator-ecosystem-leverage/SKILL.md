# 创业加速器生态杠杆

## 一句话总述

> 加速器不是"给钱的机构"，而是"杠杆放大器"——它的价值不在资源本身，而在于你能否将1份资源撬动10份效果。

## 核心框架：五层杠杆模型

### Layer 1: 资源杠杆

| 资源类型 | 杠杆倍数 | 使用要点 |
|----------|----------|----------|
| 算力 | 3-5x | 用来验证而非赖以生存 |
| 资金 | 2-3x | 买时间而非买规模 |
| 办公空间 | 1-2x | 价值在于物理接触密度 |

### Layer 2: 认知杠杆

课程选择原则：选你当前阶段最缺的，课后48小时内应用到自己的项目。

### Layer 3: 网络杠杆（最高ROI）

1. **同届网络**：互为第一用户和反馈源
2. **校友网络**：找"比你早6个月"的人
3. **投资人网络**：先建立"被记住"的印象

### Layer 4: 信号杠杆

进入加速器本身就是强信号——1%录取率>5%>20%>无筛选。

### Layer 5: 飞轮杠杆

你的成功放大加速器品牌，加速器回馈更多资源。主动贡献（分享经验、引荐人才）。

## 📋 30-60-90天行动地图

### 入选后前30天：认知补位期

| 周 | 行动 | 目标 |
|----|------|------|
| W1 | 完成所有onboarding，认识全部同届创始人 | 建立"谁知道什么"的心智地图 |
| W2 | 上最适合你的课程，做能力短板诊断 | 明确自己最缺的3个能力 |
| W3 | 与3位跨期校友深度交流（找"比你早6个月"的人） | 获得可落地的实战建议 |
| W4 | 完成MCV或产品方向调整 | 用加速器反馈修正假设 |

### 31-60天：网络杠杆期

| 周 | 行动 | 目标 |
|----|------|------|
| W5-6 | 与同届最强项目建立深度连接 | 互为用户/反馈源/合作方 |
| W7 | 主动分享你的领域洞察（写一篇内部分享） | 建立专业影响力 |
| W8 | 申请使用算力/资源完成关键实验 | 用资源验证核心假设 |

### 61-90天：路演冲刺期

| 周 | 行动 | 目标 |
|----|------|------|
| W9-10 | 准备路演材料，预演5次以上 | pitch精炼到3分钟/5分钟/10分钟 |
| W11 | 路演日：深度交流>广泛撒网 | 获得高质量反馈+投资人连接 |
| W12 | 48小时内跟进所有联系，整理反馈 | 将反馈转化为行动清单 |

## 📋 路演结构模板

### 3分钟版（快速吸引）

`
[10秒] 钩子：一个反直觉的事实或数据
[30秒] 痛点：谁在什么场景下有多痛
[60秒] 方案：你怎么解决，为什么你能
[40秒] 验证：已有什么进展/数据
[20秒] 愿景：终局是什么
[20秒] 求助：你需要什么（人才/客户/资金）
`

### 5分钟版（深度交流）

`
[15秒] 钩子
[45秒] 痛点+现有方案为什么不够
[90秒] 方案+技术/模式差异点
[60秒] 验证：付费客户/MOU/数据
[45秒] 市场+时机：为什么是现在
[30秒] 团队：为什么是我们
[15秒] 求助
`

### 10分钟版（正式路演）

在5分钟版基础上扩展：
- 加入竞争分析
- 加入商业模式和单位经济
- 加入融资计划和里程碑
- 加入demo演示

## 路演日策略

**路演前（2周）：**
- [ ] 准备3个版本路演材料
- [ ] 预演至少5次，每次找人挑刺
- [ ] 提前研究到场投资人，标记5个最匹配的

**路演中（当天）：**
- [ ] 用1句话钩子吸引投资人驻足
- [ ] 深度交流>广泛撒网
- [ ] 记录每个反馈，尤其是尖锐的批评

**路演后（1周内）：**
- [ ] 48小时内跟进所有有意义的联系
- [ ] 整理反馈，识别共性问题
- [ ] 调整pitch和产品方向

## 创业方法论本土化

中国市场三个关键差异：
1. **迭代更快**：2周验证而非2月
2. **Research Founder多**：需要更多商业化补课
3. **融资环境**：天使轮更小，A轮要求更高
