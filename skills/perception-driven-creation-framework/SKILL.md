# 感知力驱动创作框架

## 一句话总述

> 你的生命经验就是最好的创作素材库——不需要凭空创造，只需要重新"看见"你已经拥有的。

## 核心洞察

### 为什么有的作品打动人，有的只是"好看"？

| 维度 | 好看但不打动 | 打动人 |
|------|-------------|--------|
| 来源 | 技术驱动（我会这个技法） | 感知驱动（我看见了什么） |
| 深度 | 表面审美 | 生命经验 |
| 共鸣 | "哇，好看" | "这个懂我" |
| 记忆 | 看过就忘 | 久久回味 |
| 复制 | 容易被AI批量生产 | AI无法复制你的经历 |

### 感知力的三个层次

1. **表层感知**：看到现象（"人们在加班"）
2. **深层感知**：感受到情绪（"无奈但还在坚持"）
3. **根源感知**：理解结构性原因（"这是这个时代的生存状态"）

→ 创作力 = 从深层和根源感知中提炼象征的能力

## 四阶创作路径

### Phase 1: 感知力挖掘

**📋 根源文化挖掘工作表：**

| 问题 | 我的回答 |
|------|---------|
| 我在哪里长大？那个地方有什么独特的？ | _________ |
| 我小时候最深刻的记忆是什么？ | _________ |
| 我的家庭有什么与众不同的习惯/传统？ | _________ |
| 我经历过最大的困难是什么？它教会了我什么？ | _________ |
| 我最不愿意让别人看到的我是什么样？ | _________ |
| 我身边有什么被忽视但我很珍视的东西？ | _________ |

> 优秀创作者往往从自己的成长经历、文化背景中汲取灵感——这部分内容最具价值和吸引力。缺乏根源性，思想容易变得标准化。

**感知力增强练习：**
- **日常3问**：每天问自己——今天什么让我惊讶？什么让我感动？什么让我困惑？
- **视角切换**：用5种不同身份看同一个场景（老人/小孩/外国人/动物/外星人）
- **情感考古**：翻一张老照片，回忆当时的心情和没说出口的话

### Phase 2: 故事提炼

**从感知到故事的公式：**

`
真实感知 + 情感反应 + 未被表达的声音 = 创作故事内核
`

**故事提炼三问：**
1. 我感知到的核心真相是什么？
2. 为什么没有人说出这个真相？
3. 如果用一个角色/形象来代表这个真相，它会是什么？

**范例解析：**

| 真实感知 | 情感反应 | 未被表达的声音 | 故事内核 | 象征IP |
|----------|---------|--------------|---------|--------|
| 职场人天天抱怨内卷 | 无奈但坚韧 | "我不想认输但真的好累" | 小强的生命力 | 蟑螂姐 |
| 宠物离世 | 思念+愧疚 | "我没能好好照顾你" | 陪伴与遗憾 | 鹦鹉系列 |
| 狗看起来凶但很温柔 | 反差的温暖 | "别被外表骗了" | 刚柔并济 | 德牧反差 |

### Phase 3: 象征构建

**反差感——最强大的创作技法：**

反差感 = 外在形象 + 内在真相 的张力

| 外在 | 内在 | 张力 | 效果 |
|------|------|------|------|
| 丑陋（蟑螂） | 坚韧（踩不死） | 生存意志 | 幽默+共鸣 |
| 凶猛（德牧） | 温柔（舔手） | 刚柔并济 | 意外+治愈 |
| 漂亮（鹦鹉） | 脆弱（已离去） | 生命无常 | 伤感+珍惜 |
| 强大（AI） | 依赖（需人类引导） | 人机关系 | 反思+前瞻 |

**反差感构建模板：**

`
1. 选择一个载体（动物/物品/场景）
2. 定义它的"大众印象"（人们通常怎么看它）
3. 找到它的"隐藏真相"（你观察到什么别人没看到的）
4. 张力 = 大众印象 vs 隐藏真相
5. 用这个张力作为IP的核心矛盾
`

### Phase 4: IP成型

**从象征到完整IP的扩展路径：**

`
核心矛盾 → 角色设计 → 世界观 → 产品系列 → 传播策略
`

**IP扩展矩阵：**

| 层级 | 内容 | 载体 |
|------|------|------|
| L1 核心角色 | 主角设计+性格设定 | 手绘角色卡 |
| L2 情绪表达 | 角色在不同场景下的反应 | 表情包/贴纸 |
| L3 故事线 | 角色的日常/冒险 | 漫画/短片 |
| L4 延展产品 | 角色进入生活 | 文创/周边 |
| L5 社区共鸣 | 他人与角色产生连接 | 社交传播/同人 |

## 个人风格识别工具

> 用AI生成的东西千篇一律，因为缺乏"你"在里面。风格就是"你"的视觉指纹。

**风格三问：**
1. 我反复被什么类型的视觉/叙事吸引？（暗色/明亮？简洁/繁复？幽默/深沉？）
2. 我自然地（不经思考地）会怎么画/写/说？（第一反应就是你的风格底色）
3. 别人看到我的作品会用什么词形容？（问问3个朋友）

**风格测试法：**
1. 用AI生成10张不同风格的作品
2. 选出最"不像我"的3张和最"像我"的3张
3. 分析差异——最"像我"的那些共同点就是你的风格方向

## AI时代的创作者心态

### AI是创意放大器，不是替代者

| 心态 | 表现 | 结果 |
|------|------|------|
| 工具依赖 | 让AI主导创作 | 通用审美，无辨识度 |
| 工具辅助 | 我主导，AI执行 | 个人风格+效率提升 |
| 创意伙伴 | 我与AI对话碰撞 | 超越个人的创新可能 |

### 关键原则

1. **先有自己的想法，再让AI放大**——没有输入的AI输出是噪音
2. **根源文化是最好的差异化**——AI无法复制你的成长经历
3. **归属感来自投入深度**——你对创作过程的参与度决定了"这是不是你的"
4. **语文能力是驾驭AI的前提**——清晰表达=有效Prompt=更好的AI输出

## 自检清单

- [ ] 我的创作是否从真实感知出发（而非从技术/工具出发）？
- [ ] 我是否挖掘了自己的根源文化作为灵感来源？
- [ ] 我的作品是否有"反差感"的核心张力？
- [ ] 我是否能说出"这不是AI能独自生成的"的理由？
- [ ] 我的个人风格是否可识别（问3个朋友能说对吗）？
- [ ] 我是否在用AI辅助而非依赖AI？
