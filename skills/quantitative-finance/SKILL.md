# 量化金融分析技能

## 技能概述
为金融产品（基金、股票、ETF等）的量化分析提供系统化框架，涵盖数据获取、指标计算、风险评估、对比分析、策略实现等核心环节。

**策略整合**：本技能与 `quantitative-strategy-framework` 配合使用，提供完整的量化交易策略库。

## 核心分析维度

### 1. 基本信息维度
- 产品全称、代码、成立日期
- 基金类型（指数型/主动型/增强型）
- 管理人、基金经理及任职时间
- 最新规模、份额变化趋势
- 费率结构（管理费、托管费、申赎费）

### 2. 跟踪标的与策略维度
**指数型基金：** 标的指数、跟踪误差、跟踪偏离度、增强策略类型
**主动型基金：** 投资策略、选股逻辑与行业偏好、换手率水平

### 3. 业绩表现维度
**收益指标：** 各时间段收益率、年化收益、超额收益
**风险指标：** 最大回撤、波动率、下行波动率、Beta
**风险调整收益：** 夏普比率、卡玛比率、索提诺比率、信息比率

### 4. 持仓结构维度
- 前十大重仓股及占比
- 行业分布、持仓集中度、换手率
- 资产配置比例

### 5. 流动性与交易维度
- 日均成交额、买卖价差、折溢价率
- 申赎费率阶梯、大额申购限制

### 6. 对比分析框架
- 基准一致性检验
- 费率性价比
- 业绩归因分析
- 风险收益象限
- 极端行情表现

## 量化评分模型
```
综合评分 = 收益得分*40% + 风险控制得分*30% + 费率得分*15% + 流动性得分*15%
```

## 数据获取方法
1. 公开网站（天天基金网、蛋卷基金、晨星网）
2. 金融数据接口（Tushare Pro、AkShare）
3. 手动整理（季报、年报PDF）

## 分析报告模板
标准对比报告结构：产品概览 → 费率对比 → 业绩对比 → 持仓分析 → 增强策略评估 → 综合评价

## 关键注意事项
1. 数据时效性：优先使用最新季报数据
2. 可比性原则：确保基准、类型具有可比性
3. 幸存者偏差：关注基金是否经历过合并、转型
4. 风险披露：历史业绩不代表未来

## 适用对象
量化投资者、金融分析师、基金经理
