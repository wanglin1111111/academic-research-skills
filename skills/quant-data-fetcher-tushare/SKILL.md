# quant-data-fetcher-tushare

## Skill Overview
Use tushare to fetch financial data.

## Installation
```bash
pip install tushare
```

## Usage Example
```python
import tushare as lib

# Fetch data example
data = lib.get_data(...)
```

## Features
- Fetch historical data
- Fetch real-time quotes
- Data cleaning and preprocessing
- Save data (CSV, Excel)

## Test

