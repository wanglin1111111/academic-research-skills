# 投资失误知识库技能

## 技能概述
基于真实失败案例，帮助用户识别投资陷阱、避免常见失误，建立系统性风险意识。

## 核心能力

### 1. 失败案例结构化存储
- 案例描述、失误类型、损失程度
- 关键决策点、替代方案
- 经验教训、防范措施

### 2. 失误模式识别
常见失误类型：
- 追涨杀跌（情绪驱动）
- 重仓单股（仓位管理失误）
- 忽视基本面（投机心态）
- 杠杆爆仓（风险管理缺失）
- 盲信消息（信息验证不足）

### 3. 风险预警信号
- 异常成交量放大
- 高管频繁减持
- 财报审计意见异常
- 行业政策突变

### 4. 案例检索与匹配
根据当前投资标的，检索相似历史案例

### 5. 投资决策检查清单
- 基本面是否扎实？
- 估值是否合理？
- 仓位是否可控？
- 止损是否明确？

## 使用示例
```
检索"追涨杀跌"相关失败案例
分析XX股票的风险预警信号
生成投资决策检查清单
```

## 适用对象
股票投资者、基金经理、风险管理人员
