# 股票分析技能

## 技能概述
提供个股深度分析框架，涵盖基本面、技术面、资金面、行业对比等维度。支持自动化数据获取（AKShare）和缠论图表分析。

## 核心分析维度

### 0. 量化策略选择指南

**根据市场环境选择策略类型**：

| 市场环境 | 推荐策略类型 | 具体策略 |
|---------|-------------|----------|
| 趋势明确 | 趋势跟踪 | ATR通道突破、MACD共振 |
| 震荡市场 | 均值回归 | 布林带收缩、RSI极端值 |
| 价格差异 | 套利 | 跨交易所套利、ETF折溢价 |
| 特殊事件 | 事件驱动 | 财报波动率、指数调整 |
| 未知/混合 | 组合策略 | 动态风险平价、多策略组合 |

**详细策略框架**：参考 `quantitative-strategy-framework/SKILL.md`

### 1. 基本面分析
- 财务数据：营收、净利润、ROE、毛利率
- 估值指标：PE、PB、PS、PEG
- 成长性：营收增长率、净利润增长率
- 财务健康度：资产负债率、现金流

### 2. 技术面分析

#### 2.1 传统技术指标
- 趋势判断：均线系统、MACD、布林带
- 动量指标：RSI、KDJ
- 成交量分析：量价关系
- 支撑阻力位

#### 2.2 缠论分析（图表分析）
**核心理念**：将数据当成图形来分析，分为上升趋势、下降趋势、震荡三种状态

**核心概念**：
- 笔：最基本的价格运动单位
- 线段：由笔构成的有向线段
- 中枢：价格震荡整理区域
- 走势类型：上涨、下跌、盘整

**分析方法**：
- 趋势判断：通过线段方向识别趋势
- 买卖点：一类买点（趋势底）、二类买点（回调底）、三类买点（突破）
- 背驰判断：价格创新高/新低，但动能指标不创新高/新低
- 区间套：大周期决定小周期，小周期验证大周期

**数据获取方式**：
- 传统方式：结构化数据（CSV、API）
- 缠论方式：图表截图（更符合图像思维）
- 推荐工具：腾讯财经、东方财富网页截图

### 3. 资金面分析

### 4. 行业对比

## 数据来源

### 自动化数据获取（推荐）
- **AKShare**：开源免费，支持A股、港股、美股、期货等
  - 安装：`pip install akshare`
  - 示例：`ak.stock_zh_a_hist(symbol="000001", period="daily")`
  - 优势：免费、数据全面、更新及时

- **Tushare Pro**：需要注册获取token，部分数据需要积分
  - 适合：专业量化分析
  - 劣势：有使用门槛

- **Yahoo Finance API**：国际股票数据
  - 适合：美股、港股分析

### 传统数据源
- 东方财富、同花顺
- Wind、Choice
- 公司财报、公告

### 图表数据（缠论分析用）
- 腾讯财经：https://gu.qq.com/
- 东方财富：http://quote.eastmoney.com/
- 雪球：https://xueqiu.com/
