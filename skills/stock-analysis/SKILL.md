# 股票分析技能

## 技能概述
提供个股深度分析框架，涵盖基本面、技术面、资金面、行业对比等维度。

## 核心分析维度

### 1. 基本面分析
- 财务数据：营收、净利润、ROE、毛利率
- 估值指标：PE、PB、PS、PEG
- 成长性：营收增长率、净利润增长率
- 财务健康度：资产负债率、现金流

### 2. 技术面分析
- 趋势判断：均线系统、MACD、布林带
- 动量指标：RSI、KDJ
- 成交量分析：量价关系
- 支撑阻力位

### 3. 资金面分析
- 主力资金流向
- 北向资金持仓变化
- 机构调研频次
- 大股东增减持

### 4. 行业对比
- 同业估值对比
- 市场份额变化
- 竞争格局分析

### 5. 风险提示
- 政策风险
- 行业周期风险
- 公司特有风险

## 分析报告模板
1. 公司概况
2. 财务分析
3. 估值分析
4. 技术面分析
5. 资金面分析
6. 投资建议（买入/持有/卖出）
7. 风险提示

## 数据来源
- 东方财富、同花顺
- Wind、Choice
- 公司财报、公告

## 适用对象
价值投资者、技术分析师、基金经理
