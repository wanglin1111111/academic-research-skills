# 金融量化交易分析技能 (Quantitative Finance Analysis Skill)

## 技能概述
本技能为金融产品（基金、股票、ETF等）的量化分析提供系统化框架，涵盖数据获取、指标计算、风险评估、对比分析等核心环节。

---

## 核心分析维度

### 1. 基本信息维度
**必备要素：**
- 产品全称、代码、成立日期
- 基金类型（指数型/主动型/增强型）
- 管理人、基金经理及任职时间
- 最新规模、份额变化趋势
- 费率结构（管理费、托管费、申赎费）

**数据来源：**
- 天天基金网、蛋卷基金、晨星网
- 基金公司官网、定期报告
- Wind、Choice等金融终端

---

### 2. 跟踪标的与策略维度
**指数型基金关键点：**
- 标的指数全称、编制方案
- 跟踪误差（Tracking Error）
- 跟踪偏离度（Tracking Difference）
- 增强策略类型（量化多因子/Smart Beta/打新等）

**主动型基金关键点：**
- 投资策略描述
- 选股逻辑与行业偏好
- 换手率水平

---

### 3. 业绩表现维度
**收益指标：**
- 近1月/3月/6月/1年/3年/5年/成立以来收益率
- 年化收益率
- 超额收益（相对基准）
- 分年度收益表现

**风险指标：**
- 最大回撤（Maximum Drawdown）
- 波动率（标准差）
- 下行波动率（Downside Deviation）
- 贝塔系数（Beta）

**风险调整收益：**
- 夏普比率（Sharpe Ratio）= (年化收益 - 无风险利率) / 波动率
- 卡玛比率（Calmar Ratio）= 年化收益 / 最大回撤
- 索提诺比率（Sortino Ratio）= (年化收益 - 无风险利率) / 下行波动率
- 信息比率（Information Ratio）= 超额收益 / 跟踪误差

---

### 4. 持仓结构维度
**股票持仓：**
- 前十大重仓股及占比
- 行业分布（申万一级/二级）
- 持仓集中度（前10持仓占比）
- 换手率与调仓频率

**资产配置：**
- 股票/债券/现金比例
- 港股通持仓占比
- 衍生品使用情况

---

### 5. 流动性与交易维度
**场内基金（LOF/ETF）：**
- 日均成交额
- 买卖价差（Bid-Ask Spread）
- 折溢价率
- 做市商情况

**场外基金：**
- 申购赎回到账时间
- 大额申购限制
- 赎回费率阶梯

---

### 6. 对比分析框架
**同类基金对比：**
1. **基准一致性检验**：确认是否跟踪同一指数
2. **费率性价比**：综合费率 vs 超额收益
3. **业绩归因分析**：
   - 选股贡献 vs 择时贡献
   - 行业配置差异
   - 增强策略有效性
4. **风险收益象限**：绘制收益-波动率散点图
5. **极端行情表现**：牛市/熊市/震荡市分别表现

**量化评分模型：**
```
综合评分 = 收益得分×40% + 风险控制得分×30% + 费率得分×15% + 流动性得分×15%

收益得分 = (近1年收益排名 + 近3年收益排名) / 2
风险控制得分 = (夏普比率排名 + 最大回撤排名) / 2
费率得分 = 100 - (综合费率 / 同类平均费率) × 100
流动性得分 = 根据成交额/折溢价率分档打分
```

**收益表现：**
- 各时间段收益率对比表

- 明确推荐或中性观点
- 配置比例建议
- 风险提示

## 关键注意事项

1. **数据时效性**：优先使用最新季报数据，注明数据截止日期
2. **可比性原则**：确保对比基金的基准、类型具有可比性
3. **幸存者偏差**：关注基金是否经历过合并、转型
4. **市场环境**：说明分析期间的市场风格（成长/价值/周期）
5. **客观中立**：避免主观臆断，用数据说话
6. **风险披露**：明确历史业绩不代表未来，投资有风险

### 计算最大回撤
```python
def max_drawdown(net_values):
    """计算最大回撤"""
    cummax = net_values.cummax()
    drawdown = (net_values - cummax) / cummax
    return drawdown.min()
```
