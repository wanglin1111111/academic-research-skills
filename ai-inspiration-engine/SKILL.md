# AI Inspiration Engine

> AI灵感共创引擎专家 - 提供AI驱动的创意生成与产品孵化能力

## 📋 概述

本技能是一个AI+创新工具包，帮助创新者、产品经理、创业者利用AI技术进行创意生成、想法验证和产品孵化。整合太阳协会灵感共创项目方法论，构建从创意到产品的完整闭环。

## 🎯 核心能力

### 1. AI创意生成
- 多维度头脑风暴
- 创意组合与碰撞
- 趋势洞察融合
- 创意评估与筛选

### 2. 想法孵化
- 可行性分析
- 用户需求验证
- 商业模式设计
- 最小可行方案

### 3. Demo快速验证
- 原型设计指导
- MVP构建路径
- 用户测试方案
- 数据反馈分析

### 4. 创新工作坊
- 工作坊流程设计
- 引导工具与模板
- 参与者引导技巧
- 产出整理与跟进

## 📊 知识覆盖范围

## 🧪 场景测试

### 测试场景1：AI产品创意工作坊

**输入：**
- 主题：AI+个人效率工具
- 参与者：5人团队（产品/设计/开发）
- 时长：2小时

**预期输出：**
1. 趋势洞察报告（3-5个趋势方向）
2. 创意清单（20+个产品创意）
3. 可行性评估矩阵
4. Top 3优选方案
5. 每个方案的MVP计划

**验证标准：**
- 创意数量充足，质量多样
- 评估维度合理，标准一致
- 优选方案有差异化
- MVP计划可执行

### 测试场景2：想法验证流程

**输入：**
- 创意：AI驱动的个人财务助手
- 假设：用户需要自动记账和智能建议
- 目标用户：年轻职场人

**预期输出：**
1. 验证实验设计
2. 用户访谈提纲（10个问题）
3. MVP功能清单（3个核心功能）
4. 成功指标定义
5. 下一步行动计划

**验证标准：**
- 假设可测试、可证伪
- 访谈问题有针对性
- MVP范围合理不臃肿
- 指标具体可衡量

### 测试场景3：Demo快速构建

**输入：**
- 需求：AI写作助手Demo
- 功能：输入主题→生成大纲→完善段落
- 技术栈：前端+AI API

**预期输出：**
1. 技术选型建议
2. 开发计划（7天）
3. 核心代码框架
4. 测试用例
5. 演示准备清单

**验证标准：**
- 技术方案可行
- 开发计划合理
- 代码结构清晰
- Demo可演示核心价值

## 📈 商业化路径

### 阶段1：服务验证（0-3个月）
- 目标：验证创新陪跑服务价值
- 服务：创意工作坊 + 1v1咨询
- 收费：工作坊2999元/场，咨询500元/时

### 阶段2：产品打磨（3-6个月）
- 目标：标准化服务流程
- 服务：创新加速营
- 收费：9999元/期（4周）

### 阶段3：规模增长（6-12个月）
- 目标：建立创新服务品牌
- 服务：企业创新内训
- 收费：定制方案（5-20万/项目）

### 阶段4：平台化（12个月+）
- 目标：创新资源对接平台
- 模式：服务+平台+投资
- 扩展：创意交易、创新大赛、孵化器

## ⚠️ 注意事项

1. **创意密度**：好的想法来自大量想法
2. **快速失败**：低成本验证，快速迭代
3. **用户中心**：始终围绕用户痛点
4. **团队匹配**：创意与团队能力匹配
