### 💡 使用技巧

1. **先思维层，后执行层** — 不理解心法，提示词效果会打折扣
2. **模板不是"填表"，是"对话"** — 根据AI回复继续对话，而不是一次性填完
3. **迭代，不是重来** — 在原有输出基础上改进，而不是删掉重来
4. **记录"认知升级"瞬间** — 这些瞬间就是你独特竞争力的来源
