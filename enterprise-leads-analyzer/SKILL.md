# Enterprise Leads Analyzer

> 企业线索分析专家 - 提供企业画像、商业模式分析、投资标的筛选等专业服务

## 📋 概述

本技能是一个企业信息分析工具，帮助用户快速获取企业画像、进行竞品对标分析、筛选投资标的、构建销售线索。适用于投资研究、商业分析、销售拓展等场景。

## 🎯 核心能力

### 1. 企业画像分析
- 企业基本信息查询（名称、规模、业务范围）
- 核心技术梳理
- 团队背景分析
- 发展历程追踪

### 2. 竞品对标分析
- 同行业企业对比
- 技术路线差异分析
- 商业模式对比
- 市场定位分析

### 3. 投资标的筛选
- 行业赛道筛选
- 融资阶段匹配
- 技术门槛评估
- 市场潜力判断

### 4. 销售线索构建
- 目标客户画像
- 决策链路分析
- 合作切入点识别
- 谈判策略建议

## 📊 数据覆盖范围

本技能依赖的企业数据库包含以下维度的企业信息：

| 维度 | 说明 |
|------|------|
| 企业描述 | 业务范围、发展历程、核心产品 |
| 企业画像 | 规模、团队、融资、市场地位 |
| 核心技术 | 技术栈、专利、技术壁垒 |
| 对标企业 | 国内外同行、竞品分析 |
| 上下游企业 | 供应链、客户群、合作伙伴 |
| 生态合作 | 周边生态、合作逻辑、战略布局 |
| 商业模式 | 盈利模式、收入来源、成本结构 |

## 🚀 使用方法

### 典型应用场景

**场景1：企业基本信息查询**
```
用户："帮我分析一下做AI新药开发的公司有哪些？"
执行：搜索关键词 "AI 新药开发"，整理企业列表和关键信息
```

**场景2：竞品对比分析**
```
用户："对比一下国内外创业方向差异"
执行：搜索关键词 "创业方向"、"国内外对比"，生成对比报告
```

**场景3：投资标的研究**
```
用户："推荐一些AI硬件领域的投资标的"
执行：搜索 "AI硬件"、"机器人"、"芯片" 等关键词，筛选符合投资标准的企业
```

**场景4：销售线索筛选**
```
用户："帮我找一些做3D打印的潜在客户"
执行：搜索 "3D打印"，提取企业联系方式、决策人、业务规模等信息
```

### 推荐问题示例

- 利用人工智能技术做新药开发的企业有哪些？
- 分析国内外创业方向差异
- 海外企业最新创新创业方向
- 3D打印领域的投资标的推荐
- AI大模型赛道的企业画像

## 📝 输出格式

分析结果以结构化格式输出，包括：

1. **企业列表**：符合条件的企业清单
2. **关键信息**：每个企业的核心数据点
3. **对比分析**：横向对比表格（适用于竞品分析）
4. **建议总结**：基于数据的洞察和建议

### 示例输出

```markdown

| 企业B | AI新药开发 | A轮 | 分子生成模型 | 新兴企业 |

### 关键洞察
1. 该领域主要集中在XX方向
2. 技术壁垒主要来自XX
3. 市场机会在于XX
```

## ⚙️ 技术实现

### 数据源配置

本技能需要配置企业数据源。支持以下数据源类型：

1. **知识库API**：通过知识库搜索接口获取企业信息
2. **本地数据库**：连接本地企业数据库
3. **第三方API**：集成企业信息查询API

### 配置示例

```yaml
# config.yaml
data_source:
  type: knowledge_base  # 或 local_db / third_party_api
  knowledge_base:
    id: "your-kb-id"  # 替换为实际知识库ID
    api_endpoint: "https://api.example.com/search"
  local_db:
    path: "/path/to/enterprise.db"
  third_party_api:
    provider: "tianyancha"  # 或其他企业信息API
    api_key: "${API_KEY}"
```
