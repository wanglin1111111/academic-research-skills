# AI Personal Knowledge

> AI+个人知识管理专家 - 提供AI驱动的个人知识体系构建能力

## 📋 概述

本技能是一个AI+个人知识管理工具包，帮助知识工作者、学习者构建系统化的个人知识体系。整合第二大脑方法论、AI笔记工具、知识图谱等先进理念与技术。

## 🎯 核心能力

### 1. 知识体系设计
- 个人知识架构规划
- 领域知识树构建
- 知识分类与标签体系
- 知识生命周期管理

### 2. AI辅助笔记
- 智能笔记摘要
- 自动标签与分类
- 知识卡片生成
- 关联推荐

### 3. 学习路径规划
- 技能图谱构建
- 学习资源推荐
- 进度追踪与调整
- 知识缺口分析

### 4. 跨平台聚合
- 多源信息同步
- 统一知识检索
- 数据可移植性
- 备份与恢复

## 📊 知识覆盖范围

| 场景 | 核心功能 | 技术方案 | 适用人群 |
|------|---------|---------|---------|
| 知识体系 | 架构设计、分类规划 | 知识图谱 + 本体论 | 知识工作者 |
| AI笔记 | 摘要、分类、关联 | LLM + 向量数据库 | 学生、研究者 |
| 学习规划 | 路径设计、资源推荐 | 推荐系统 + 知识追踪 | 终身学习者 |
| 知识检索 | 语义搜索、智能问答 | RAG + Embedding | 所有人 |

## 📝 输出格式

### 个人知识体系设计方案

```markdown
# 个人知识体系设计

## 核心原则
- **原子化**：每条笔记聚焦一个概念
- **关联化**：建立知识间的联系
- **层次化**：从具体到抽象
- **动态化**：持续迭代更新

## 领域划分
1. **专业领域**（核心收入来源）
   - [领域1]：AI产品设计
   - [领域2]：用户研究

2. **兴趣领域**（长期积累）
   - [领域1]：心理学
   - [领域2]：写作

3. **通用领域**（底层能力）
   - [领域1]：项目管理
   - [领域2]：沟通表达

## 知识层级
```
L1：原始笔记（读书摘录、课程笔记、会议记录）
  ↓ 加工
L2：知识卡片（概念解释、方法总结、案例记录）
  ↓ 关联
L3：知识体系（专题文档、方法论、心智模型）
  ↓ 应用
L4：输出成果（文章、课程、产品、决策）
```

## 工具栈推荐
- 收集：Readwise、Instapaper、微信读书
- 整理：Obsidian、Notion、Logseq
- 连接：Roam Research、Heptabase
- 输出：Obsidian Publish、Notion Site
```

### AI笔记工作流

```markdown
# AI辅助笔记工作流

## 阶段1：信息捕获
- 多渠道聚合（网页、PDF、音频、视频）
- 一键保存到统一收件箱
- 元数据自动记录（来源、时间、标签）

## 阶段2：智能处理
1. 自动摘要（100字→300字→500字）
2. 关键概念提取
3. 自动标签建议
4. 相似内容推荐

## 阶段3：知识组织
- 自动分类到对应领域
- 建立双向链接
- 生成知识图谱
- 识别知识缺口

## 阶段4：价值输出
- 智能问答
- 内容生成
- 复习提醒
- 知识分享
```

## 🧪 场景测试

### 测试场景1：读书笔记系统

**输入：**
- 书籍：《原子习惯》
- 阅读方式：电子书标注
- 目标：建立可复用的习惯方法论

**预期输出：**
1. 章节摘要卡片（20-30张）
2. 核心方法论提取（4定律）
3. 与现有知识的关联
4. 个人行动计划模板
5. 复习提醒设置

**验证标准：**
- 摘要准确反映原著观点
- 方法论可落地执行
- 关联自然合理
- 笔记可复用性强

### 测试场景2：AI学习路径规划

**输入：**
- 学习目标：从零基础到AI产品经理
- 当前水平：了解基础概念
- 可用时间：每周10小时
- 周期：6个月

**预期输出：**
1. 技能图谱（AI产品经理能力模型）
2. 分阶段学习计划（基础/进阶/实战）
3. 每周学习任务清单
4. 推荐资源清单（书/课/社区）
5. 里程碑与评估标准

**验证标准：**
- 路径阶梯合理，无跳跃
- 资源质量高且可获得
- 任务可执行且不超载
- 评估标准具体可衡量

### 测试场景3：跨平台知识聚合

**输入：**
- 数据源：微信读书、知乎、B站、GitHub、Notion
- 内容类型：文章、视频、代码、笔记
- 时间跨度：1年积累

**预期输出：**
1. 统一知识库（所有来源聚合）
2. 自动分类与标签
3. 知识关联图谱
4. 检索与问答能力
5. 数据可导出（标准格式）

**验证标准：**
- 数据迁移完整无丢失
- 分类准确率达80%以上
- 关联推荐有启发意义
- 检索响应快速准确

## ⚠️ 注意事项

1. **数据主权**：用户数据可导出、可迁移
2. **隐私保护**：本地优先，云端可选
3. **工具简单**：避免功能过重
4. **长期主义**：知识管理是终身习惯
