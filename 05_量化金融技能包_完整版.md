# 05 量化金融技能包（完整版）

## 概述
金融产品（基金、股票、ETF）的量化分析系统化框架

## 六大分析维度

### 1. 基本信息维度
产品全称、代码、成立日期、基金类型、管理人、最新规模、费率结构

### 2. 跟踪标的与策略维度
标的指数、跟踪误差、跟踪偏离度、增强策略类型

### 3. 业绩表现维度
**收益指标**：各时间段收益、年化收益、超额收益
**风险指标**：最大回撤、波动率、下行波动率、贝塔系数
**风险调整收益**：夏普比率、卡玛比率、索提诺比率、信息比率

### 4. 持仓结构维度
前十大重仓股、行业分布、持仓集中度、换手率

### 5. 流动性与交易维度
日均成交额、买卖价差、折溢价率、申赎费率阶梯

### 6. 对比分析框架
- 基准一致性检验
- 费率性价比
- 业绩归因分析
- 风险收益象限
- 极端行情表现

## 量化评分模型
```
综合评分 = 收益得分×40% + 风险控制得分×30% + 费率得分×15% + 流动性得分×15%
```

## 数据获取方法
- 天天基金网、蛋卷基金、晨星网
- Tushare Pro、AkShare
- 基金季报、年报PDF
