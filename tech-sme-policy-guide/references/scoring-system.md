# 科技型中小企业评分系统详解

## 评分体系概述

科技型中小企业评价指标满分100分，由三项指标加权计算得出：
- **科技人员指标**：20分（一票否决项）
- **研发投入指标**：50分（核心指标）
- **科技成果指标**：30分

**重要规则**：
1. 科技人员指标得0分，直接一票否决，无论总分多高都无法入库
2. 研发投入指标系统自动选择两种计算方式中的较高分值
3. 总分≥60分即可通过评价

## 一、科技人员指标（20分）- 一票否决

### 评分标准

| 科技人员占比 | 得分 |
|-------------|------|
| ≥30% | 20分 |
| ≥25%且<30% | 16分 |
| ≥20%且<25% | 12分 |
| ≥15%且<20% | 8分 |
| ≥10%且<15% | 4分 |
| <10% | 0分（一票否决）|

### 科技人员定义

**包括**：
1. 直接从事研发和相关技术创新活动的人员
2. 专门从事研发活动的管理人员
3. 提供直接技术服务的人员

**不包括**：
1. 仅提供行政支持、财务、人事等间接服务的人员
2. 从事生产制造、销售、一般管理的人员
3. 临时聘用的非研发人员

### 统计口径

**职工总数计算**：
- 企业在职人员
- 兼职人员（全年须在企业累计工作183天以上）
- 临时聘用人员（全年须在企业累计工作183天以上）

**科技人员计算**：
- 同上统计口径
- 需提供劳动合同、社保缴纳证明、工资发放记录等证明材料

### 常见问题

**Q1：兼职人员可以算科技人员吗？**
A：可以，但需满足全年累计工作183天以上，且实际从事研发活动。

**Q2：科技人员占比刚好10%得几分？**
A：4分。≥10%且<15%得4分。

**Q3：科技人员占比9.9%怎么办？**
A：0分，一票否决。需增加科技人员比例至10%以上。

## 二、研发投入指标（50分）- 核心指标

### 两种计算方式

系统会自动计算两种方式，选择分值较高的作为最终得分。

#### 方式一：研发费用占销售收入比例

| 研发费用/销售收入 | 得分 |
|------------------|------|
| ≥6% | 50分 |
| ≥5%且<6% | 40分 |
| ≥4%且<5% | 30分 |
| ≥3%且<4% | 20分 |
| ≥2%且<3% | 10分 |
| <2% | 0分 |

#### 方式二：研发费用占成本费用比例

| 研发费用/成本费用 | 得分 |
|------------------|------|
| ≥30% | 50分 |
| ≥25%且<30% | 40分 |
| ≥20%且<25% | 30分 |
| ≥15%且<20% | 20分 |
| ≥10%且<15% | 10分 |
| <10% | 0分 |

### 研发费用归集

**可归集的研发费用**：
1. **人员人工费用**：直接从事研发活动人员的工资薪金、五险一金
2. **直接投入费用**：研发活动直接消耗的材料、燃料、动力费用
3. **折旧费用**：研发用仪器设备折旧费
4. **无形资产摊销**：研发用软件、专利权等摊销费
5. **新产品设计费**：新产品设计、新工艺规程制定费
6. **其他相关费用**：技术图书资料费、专家咨询费等（不得超过总研发费用的10%）

**不可归集的费用**：
- 生产设备购置费（应计入固定资产）
- 普通管理人员的工资
- 销售费用、管理费用
- 财务费用

### 统计口径

**销售收入**：
- 主营业务收入
- 其他业务收入
- 视同销售收入

**成本费用支出总额**：
- 主营业务成本
- 其他业务成本
- 销售费用
- 管理费用
- 财务费用

### 优化建议

**对于销售收入较小的企业**：
- 方式二（占成本费用比例）可能得分更高
- 合理增加研发费用归集

**对于成本费用较小的企业**：
- 方式一（占销售收入比例）可能得分更高
- 控制成本费用，突出研发投入

## 三、科技成果指标（30分）

### 评分标准

| 知识产权情况 | 得分 |
|-------------|------|
| 1项及以上Ⅰ类知识产权 | 30分 |
| 4项及以上Ⅱ类知识产权 | 24分 |
| 3项Ⅱ类知识产权 | 18分 |
| 2项Ⅱ类知识产权 | 12分 |
| 1项Ⅱ类知识产权 | 6分 |
| 没有知识产权 | 0分 |

### 知识产权分类

**Ⅰ类知识产权**（含金量高）：
- 发明专利（含国防专利）
- 植物新品种
- 国家级农作物品种
- 国家新药
- 国家一级中药保护品种
- 集成电路布图设计专有权

**Ⅱ类知识产权**（含金量较低）：
- 实用新型专利
- 外观设计专利
- 软件著作权

### 认定规则

**必须同时满足**：
1. **中国境内授权**：国外专利不计分
2. **有效期内**：过期或无效知识产权不计分
3. **已授权**：申请中（未授权）不计分
4. **权属清晰**：知识产权权属人必须为申报企业

**不予认可的情形**：
- 通过购买、受赠等方式获得的知识产权，虽可计分，但需与主营业务相关
- 知识产权与主营业务无关联，可能被视为包装，影响审核

### 优化建议

**目标1：获得30分**
- 拥有至少1项Ⅰ类知识产权（发明专利）
- 或拥有4项以上Ⅱ类知识产权

**目标2：对于只有Ⅱ类知识产权的企业**
- 至少拥有1项软件著作权或实用新型专利（6分保底）
- 努力达到3-4项（18-24分）

**时间规划**：
- 发明专利授权周期长（1-3年），需提前布局
- 实用新型、软件著作权授权周期短（6-12个月），可快速补充

## 四、评分策略与优化

### 策略1：确保通过线（≥60分）

**推荐配置**：
- 科技人员指标：≥10%（4分）
- 研发投入指标：≥2%且<3%（10分）或更高
- 科技成果指标：至少1项Ⅱ类知识产权（6分）

**总分**：4+10+6=20分（不够，需优化）

**优化后**：
- 科技人员指标：≥15%（8分）
- 研发投入指标：≥3%且<4%（20分）
- 科技成果指标：2项Ⅱ类知识产权（12分）

**总分**：8+20+12=40分（仍不够）

**再优化**：
- 科技人员指标：≥20%（12分）
- 研发投入指标：≥4%且<5%（30分）
- 科技成果指标：3项Ⅱ类知识产权（18分）

**总分**：12+30+18=60分（刚好通过）

### 策略2：冲刺高分（≥80分）

**推荐配置**：
- 科技人员指标：≥30%（20分）
- 研发投入指标：≥6%（50分）
- 科技成果指标：1项Ⅰ类知识产权（30分）

**总分**：20+50+30=100分（满分）

### 策略3：低成本提升分数

**对于科技人员指标**：
- 将从事研发相关工作的人员（如技术支持、质量检验）计入科技人员
- 确保统计口径准确

**对于研发投入指标**：
- 规范研发费用辅助账，确保应归尽归
- 研发人员五险一金计入研发费用
- 研发用仪器设备折旧计入研发费用

**对于科技成果指标**：
- 快速申请实用新型专利（6-12个月授权）
- 快速登记软件著作权（1-3个月）
- 购买与企业主营业务相关的发明专利（需变更权属）

## 五、系统自动评分说明

### 评分流程

1. **企业填报**：企业在线填写数据
2. **系统计算**：系统根据填报数据自动计算得分
3. **直通车判断**：系统判断是否符合直通车条件
4. **结果输出**：显示评分结果或直通车确认结果

### 注意事项

1. **数据一致性**：填报数据应与税务申报表、财务报表一致
2. **材料真实性**：上传的附件材料应真实有效
3. **逻辑合理性**：各项数据之间应有逻辑关系，避免异常波动
4. **及时更新**：如企业情况发生变化，应及时更新信息

## 六、常见问题解答

### Q1：科技人员占比怎么计算？
**A**：科技人员占比 = 科技人员数 ÷ 职工总数 × 100%

### Q2：研发费用包括哪些内容？
**A**：包括人员人工、直接投入、折旧费用、无形资产摊销、新产品设计费、其他相关费用等。

### Q3：国外专利可以计分吗？
**A**：不可以。必须是发生在中国境内授权获取的知识产权。

### Q4：申请中的专利可以计分吗？
**A**：不可以。必须是已授权的知识产权。

### Q5：总分计算错误怎么办？
**A**：检查各项数据填报是否准确，特别是科技人员数、研发费用、知识产权数量等关键信息。
