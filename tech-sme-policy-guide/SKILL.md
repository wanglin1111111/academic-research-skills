# 科技型中小企业政策指南技能

本技能提供科技型中小企业（科小）评价政策的全面解读和申报指导，基于最新政策变化提供专业分析。

## 核心功能

### 1. 政策变化解读
当用户询问政策变化、新旧政策对比时，引用 `references/policy-changes-2026.md` 提供详细分析。

**触发词**：政策变化、2026年新政、与往年区别、主管部门变更、实地核查、抽查时间

### 2. 申报资格评估
基于六大变化中的申报条件，评估企业是否符合科小评价要求。

**评估维度**：
- 基础准入条件（注册、规模、负面清单、信用、财务）
- 综合评价指标（科技人员、研发投入、科技成果）
- 直通车条件（高企、科技奖励、研发机构、标准制定）

**触发词**：我能申报吗、符合条件吗、评分标准、直通车条件

### 3. 申报流程指导
提供从填报到入库的完整流程指导。

**关键时间节点**：
- 企业填报：6月1日-8月31日
- 省级公示：10月15日前
- 集中抽查：11月30日前

**流程步骤**：注册填报 → 企业自评 → 形式审查 → 分批公示 → 公告入库

**触发词**：申报流程、时间节点、怎么申报、填报指南

### 4. 政策红利分析
解读科小评价带来的五大核心红利。

**红利清单**：
- 研发费用加计扣除100%
- 亏损结转延长至10年
- 财政资金支持（创新资金、成果转化）
- 金融信贷支持（科创助力贷）
- 科技创新券（最高15万/年）

**触发词**：政策红利、税收优惠、资金支持、融资便利

### 5. 风险识别与规避
识别申报过程中的关键风险点并提供规避建议。

**主要风险**：
- 中介风险：严禁通过第三方中介申报
- 数据一致性：研发费用需与税务报表一致
- 突击申报风险：集中抽查比例不低于5%
- 知识产权风险：数量为0需实地核查
- 科技属性风险：严禁商贸流通企业包装入库

**触发词**：申报风险、注意事项、常见问题、被拒原因

## 工作流程

### 步骤1：需求识别
根据用户问题确定需求类型：
- 政策咨询 → 提供变化解读
- 资格评估 → 评估申报条件
- 流程指导 → 详解申报步骤
- 红利分析 → 列举政策收益
- 风险规避 → 识别潜在问题

### 步骤2：信息收集
收集评估所需关键信息：
- 企业规模（职工数、销售收入、资产）
- 知识产权情况（数量、类型、有效性）
- 研发投入（研发费用、科技人员占比）
- 信用状况（违法失信记录）
- 已有资质（高企、科技奖励、研发机构）

### 步骤3：分析评估
基于收集信息进行：
- 资格符合性评估
- 评分预估（科技人员20分+研发投入50分+科技成果30分）
- 风险点识别
- 红利匹配分析

### 步骤4：提供建议
输出结构化建议：
- 申报可行性结论
- 关键准备事项
- 时间规划建议
- 风险规避措施
- 红利最大化策略

## 重要提醒

1. **时间敏感性**：科小编号仅当年有效，需每年重新申报
2. **数据真实性**：申报数据需与税务报表一致，系统会自动比对
3. **科技属性**：单纯商贸流通、简单组装加工企业严禁包装入库
4. **中介风险**：工信部明确禁止通过第三方中介申报

## 使用建议

1. 在进行资格评估时，建议按照基础条件→评分指标→直通车条件的顺序逐步评估
2. 引用政策变化时，注意说明变化前后的对比，突出对企业的影响
3. 提供建议时，结合企业实际情况给出个性化方案
4. 强调时间节点，提醒企业尽早准备，避免错过申报窗口

## 限制说明

本技能提供的政策解读基于公开信息，具体申报要求以主管部门最新通知为准。建议在正式申报前咨询当地主管部门或专业机构。
