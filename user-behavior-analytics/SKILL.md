# User Behavior Analytics

> 用户行为分析专家 - 提供AI产品用户行为监控与数据驱动优化能力

## 📋 概述

本技能是一个用户行为分析工具包，帮助AI产品团队、独立开发者监控用户行为、分析数据、评估产品价值、提升用户LTV。整合播客中分享的用户行为观测经验，解决早期OPC用户少、难获取真实反馈的痛点。

## 🎯 核心能力

### 1. 行为数据监控
- 页面流量监控
- 点击热力图分析
- 停留时长追踪
- 转化漏斗分析

### 2. 产品价值评估
- 功能使用频率分析
- 用户满意度评估
- 功能价值排序
- 资源投入优化建议

### 3. LTV提升策略
- 用户分层运营
- 留存优化
- 付费转化提升
- 生命周期价值管理

### 4. 数据驱动迭代
- A/B测试设计
- 功能效果评估
- PR（产品需求）自动生成
- 迭代优先级排序

## 📊 知识覆盖范围

| 分析维度 | 核心指标 | 分析目的 | 适用阶段 |
|---------|---------|---------|---------|
| 流量分析 | UV/PV/停留时长 | 了解用户访问情况 | 全周期 |
| 行为分析 | 点击率/转化率 | 评估功能价值 | 全周期 |
| 留存分析 | 次日/7日/30日留存 | 评估产品粘性 | 成长期 |
| 价值分析 | ARPU/LTV/CAC | 评估商业健康度 | 成熟期 |

### 关键指标监控代码

```python
# 核心指标监控示例
class ProductAnalytics:
    def __init__(self):
        self.metrics = {}
    
    def track_event(self, user_id, event_name, properties=None):
        """追踪用户事件"""
        event = {
            'user_id': user_id,
            'event': event_name,
            'timestamp': datetime.now(),
            'properties': properties or {}
        }
        self.save_event(event)
    
    def calculate_retention(self, cohort_date):
        """计算留存率"""
        query = f"""
        SELECT 
            DATE(created_at) as signup_date,
            COUNT(DISTINCT user_id) as cohort_size,
            COUNT(DISTINCT CASE WHEN DATEDIFF(created_at, '{cohort_date}') = 1 
                               THEN user_id END) as d1_retention,
            COUNT(DISTINCT CASE WHEN DATEDIFF(created_at, '{cohort_date}') = 7 
                               THEN user_id END) as d7_retention
        FROM users
        WHERE DATE(created_at) = '{cohort_date}'
        GROUP BY signup_date
        """
        return self.execute_query(query)
    
    def calculate_ltv(self, user_id):
        """计算用户LTV"""
        # 基于历史消费数据预测
        historical_value = self.get_historical_value(user_id)
        predicted_value = self.predict_future_value(user_id)
        return historical_value + predicted_value
    
    def generate_pr_from_data(self):
        """基于数据生成产品需求"""
        insights = self.extract_insights()
        return {
            'problem': insights['top_problem'],
            'solution': self.recommend_solution(insights),
            'impact': self.estimate_impact(insights),
            'priority': self.calculate_priority(insights)
        }
```
