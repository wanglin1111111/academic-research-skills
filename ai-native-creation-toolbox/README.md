## 💡 使用技巧

### 技巧1：先思维层，后执行层
不要跳过思维层直接看执行层。  
**原因**：执行层的提示词模板是基于思维层的心法设计的，不理解心法，提示词效果会打折扣。

### 技巧2：模板不是"填表"，是"对话"
不要把提示词当成"填空游戏"。  
**正确用法**：把提示词复制给AI，然后根据AI的回复，**继续对话**，而不是一次性填完就结束。

### 技巧3：迭代，不是重来
如果AI的输出不够好：
- ❌ 不好：删掉重来，重新描述
- ✅ 好：在原有输出基础上说"第X部分请再详细一点，具体是……"

### 技巧4：记录你的"认知升级"瞬间
每次用这些心法/模板，如果有"啊！原来如此"的瞬间，立刻记录下来。  
**原因**：这些瞬间就是你认知升级的证据，也是你独特竞争力的来源。
