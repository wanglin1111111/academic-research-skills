| 分数 | 描述 |
|------|------|

# skill.yaml - Skill配置文件

version: "1.0.0"

### Skill调用示例

```python
# Skill调用接口设计
class SkillCaller:
    def __init__(self, skill_registry):
        self.registry = skill_registry
    
    def call_skill(self, skill_name, input_data, context=None):
        """调用Skill"""
        skill = self.registry.get(skill_name)
        
        # 参数验证
        validated_input = skill.validate_input(input_data)
        
        # 执行Skill
        result = skill.execute(validated_input, context)
        
        # 结果后处理
        formatted_result = skill.format_output(result)
        
        return formatted_result
    
    def chain_skills(self, skill_chain, initial_input):
        """Skill链式调用"""
        current_output = initial_input
        results = []
        
        for skill_name in skill_chain:
            skill = self.registry.get(skill_name)
            current_output = skill.execute(current_output)
            results.append(current_output)
        
        return results
```
