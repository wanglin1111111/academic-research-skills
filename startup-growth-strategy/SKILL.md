# Startup Growth Strategy

> 创业增长策略专家 - 提供产品迭代、用户增长、内容营销全流程服务

## 📋 概述

本技能是一个创业增长策略工具包，帮助创业团队制定和执行增长计划。整合会议中关于后续工作计划的讨论，覆盖产品迭代、用户增长、内容营销、活动策划等核心议题。

### 1. 产品迭代规划
- 迭代节奏设计
- 功能优先级排序
- MVP到规模化路径
- 快速验证方法

### 2. 用户增长策略
- 增长黑客 tactics
- 渠道选择与优化
- 转化漏斗设计
- 留存提升策略

### 3. 内容营销计划
- 内容策略制定
- 多平台分发
- 品牌建设
- 社群运营

### 4. 活动策划执行
- 行业活动策划
- 产品发布活动
- 用户运营活动
- 品牌活动

## ⚙️ 技术实现要点

### 增长分析系统

```python
# 增长分析系统
class GrowthAnalytics:
    def __init__(self):
        self.funnel_stages = ['awareness', 'interest', 'signup', 'activation', 'payment', 'retention']
        self.metrics = {}
    
    def track_funnel(self, user_id, stage, properties=None):
        """追踪用户漏斗"""
        if stage not in self.funnel_stages:
            raise ValueError(f"Invalid stage: {stage}")
        
        event = {
            'user_id': user_id,
            'stage': stage,
            'timestamp': datetime.now(),
            'properties': properties or {}
        }
        self.save_event(event)
    
    def calculate_conversion_rate(self, from_stage, to_stage):
        """计算转化率"""
        from_count = self.get_stage_count(from_stage)
        to_count = self.get_stage_count(to_stage)
        
        if from_count == 0:
            return 0
        return to_count / from_count
    
    def analyze_growth(self, period='week'):
        """分析增长数据"""
        data = self.get_period_data(period)
        
        return {
            'new_users': self.count_new_users(data),
            'active_users': self.count_active_users(data),
            'retention_rates': self.calculate_retention(data),
            'conversion_rates': self.calculate_all_conversions(),
            'channel_performance': self.analyze_channels(data)
        }
    
    def identify_growth_levers(self):
        """识别增长杠杆"""
        levers = []
        
        # 分析各阶段转化率
        for i in range(len(self.funnel_stages) - 1):
            from_stage = self.funnel_stages[i]
            to_stage = self.funnel_stages[i + 1]
            conversion = self.calculate_conversion_rate(from_stage, to_stage)
            
            if conversion < 0.5:  # 转化率低于50%
                levers.append({
                    'stage': f"{from_stage} → {to_stage}",
                    'current_rate': conversion,
                    'potential': '高',
                    'recommendations': self.get_optimization_suggestions(from_stage, to_stage)
                })
        
        return sorted(levers, key=lambda x: x['potential'], reverse=True)
```

# 内容营销自动化系统
class ContentMarketingAutomation:
    def __init__(self):
        self.content_calendar = {}
        self.distribution_channels = []
    
    def create_content_calendar(self, themes, frequency, platforms):
        """创建内容日历"""
        calendar = {}
        date = datetime.now()
        
        for i in range(30):  # 30天内容计划
            date += timedelta(days=1)
            theme = themes[i % len(themes)]
            
            for platform in platforms:
                if platform.should_post(date, frequency):
                    content = self.generate_content(theme, platform)
                    calendar[date.strftime('%Y-%m-%d')] = {
                        'platform': platform.name,
                        'theme': theme,
                        'content': content
                    }
        
        return calendar
    
    def generate_content(self, theme, platform):
        """生成平台适配内容"""
        templates = {
            'wechat': {'length': 'long', 'style': 'professional'},
            'xiaohongshu': {'length': 'short', 'style': 'casual'},
            'douyin': {'length': 'very_short', 'style': 'entertaining'}
        }
        
        template = templates.get(platform.name, templates['wechat'])
        
        # 基于主题生成内容
        content = self.ai_generate(theme, template)
        return content
    
    def analyze_content_performance(self):
        """分析内容表现"""
        return {
            'top_performing': self.get_top_content(),
            'best_time': self.get_best_posting_time(),
            'best_format': self.get_best_format(),
            'audience_insights': self.get_audience_insights()
        }

## ⚠️ 注意事项

1. **数据驱动**：所有决策基于数据，而非直觉
2. **快速实验**：小步快跑，快速验证假设
3. **聚焦核心**：不要分散精力，聚焦核心增长杠杆
4. **用户价值**：增长必须建立在用户价值基础上
5. **长期视角**：避免短期主义，关注长期健康度
